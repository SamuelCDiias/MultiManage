# MultiManage

**MultiManage** é um sistema de gestão empresarial que permite gerenciar múltiplas empresas em um único painel. Os usuários podem alternar entre empresas e acessar dados específicos de cada uma, facilitando a gestão e a organização das operações de diferentes negócios.

## Tecnologias Utilizadas
- **Backend**: Laravel 11
- **Frontend**: Blade, Livewire, Tailwind CSS
- **Banco de Dados**: PostgreSQL
- **Autenticação**: Breeze com Livewire

## Funcionalidades Principais
- **Gerenciamento de múltiplas empresas**: Permite alternar entre empresas e acessar informações específicas de cada uma.
- **Controle de usuários**: Gerencie usuários associados a cada empresa e defina funções e permissões.
- **Interface amigável**: Usando **Tailwind CSS**, a interface é limpa e fácil de navegar, proporcionando uma experiência de usuário intuitiva.

## Instalação

### Pré-requisitos
- PHP 8.x ou superior
- Composer
- Node.js e NPM
- PostgreSQL
